This project analyzes Walmart sales data to identify trends and insights using Excel. The cleaned dataset was transformed into an interactive Excel dashboard that combines sales performance with key economic indicators.

The project demonstrates an end-to-end Business Intelligence workflow: data cleaning → analysis → visualization.

Dataset:
Weekly Walmart sales data with supporting economic variables:
Store
Date
Year
Month
Weekly Sales
Holiday Flag
Temperature
Fuel Price
CPI
Unemployment Rate

Tools Used:
Microsoft Excel – Data cleaning, Dashboard & visualization
GitHub – Documentation & version control

Data Preparation:
Date column was Used to create Year & Month columns
The data type of Fuel price, Cpi, & Unemployment were changed from string to integer using the Value formula.

Dashboard Highlights:
KPIs:
Total Sales: 673,721,987
Avg CPI: 171.58
Avg Temperature: 60.66°F
Avg Fuel Price: 3.36
Avg Unemployment: 8.00%

Visuals include:
Total Sales by Month
Sales & Holidays by Store
Avg Temperature, Fuel Price & CPI by Store
Unemployment Trend Over Time
Interactive slicers allow filtering by Store, Month, and Year.

Key Insights:
Sales show clear seasonal patterns
Economic indicators vary significantly across stores
External factors provide useful context for sales performance

Skills Demonstrated:
Excel data cleaning & transformation
KPI tracking
